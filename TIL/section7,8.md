
# 위젯트리

플러터에서의 위젯 트리는 UI를 렌더링(화면에 그리는)하는 데 사용되는 계층 구조를 의미한다.

### 기본구조
```
MaterialApp (루트)
└── Scaffold (기본 레이아웃)
    ├── AppBar (상단 바)
    ├── Body (본문 내용)
    └── FloatingActionButton (플로팅 버튼)
```

# Splash Screen (스플래시 스크린)

### YAML Structure 구조
YAML은 사람이 쉽게 읽고 쓸 수 있는 데이터 직렬화 언어이다.
